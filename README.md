# Early Warning of In-Hospital Deterioration within 12 Hours

This project predicts early signs of patient deterioration in hospitals using machine learning models.

## Technologies Used
Python
Machine Learning
Logistic Regression
Isolation Forest
Gradient Boosting

## Objective
To identify early warning signs of patient health deterioration and help doctors make proactive decisions.
